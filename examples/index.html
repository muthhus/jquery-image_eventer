P.rogressive I.mage L.oad Examples

1. Basic Progressive Image Load (single array)

This will walk you through a very basic p.i.l. example. With images passed as an array.

$.pil(['img1.jpg', 'img2.jpg', 'img3.jpg']);

Basic single array

2. Basic Progressive Image Load (single array w/ custom event)

This will walk you through a very basic p.i.l. example. With images passed as an array and a custom event.

$.pil(['img1.jpg', 'img2.jpg', 'img3.jpg'], 'name_of_event');

Basic single array w/ custom event

3. Basic Progressive Image Load (mutliple arrays)

Here we demonstrate how to load 'sets' of images. Each set will load and fire the default $.pil() loaded event.

$.pil({
  collections: [
    ['img1.jpg', 'img2.jpg', 'img3.jpg'],
    ['img4.jpg', 'img5.jpg', 'img6.jpg']
  ]
});

Basic mult array

4. Basic Progressive Image Load (mutliple objects)

Here we demonstrate how to load 'sets' of images. Each set will load and fire it's own event custom event.

$.pil({
  collections: [
    {files: ['img1.jpg', 'img2.jpg', 'img3.jpg'], completedEvent: 'name_of_event1'},
    {files: ['img4.jpg', 'img5.jpg', 'img6.jpg'], completedEvent: 'name_of_event2'}
  ]
});

Basic mult array w/ custom events

5. Passing a jQuery object

With images passed by thier parent.

$.pil($('#Some_Id_that_exists_in_the_DOM'));
